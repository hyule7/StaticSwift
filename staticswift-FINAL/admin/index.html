Setup required to see live data

Add these environment variables in Netlify to connect live analytics:

GA4_PROPERTY_ID=123456789 # GA4 Admin → Property Settings
GSC_SITE_URL=https://staticswift.co.uk/
GOOGLE_SA_EMAIL=service-account@project.iam.gserviceaccount.com
GOOGLE_SA_KEY=base64-encoded-private-key # base64 encode your JSON key file

Create a Google Cloud service account, enable Analytics Data API and Search Console API, add the service account email as a viewer in GA4 and Search Console, then base64-encode the JSON key file for GOOGLE_SA_KEY.
